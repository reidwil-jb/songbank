## Usage
```bash
$ git clone https://github.com/reidwil/songbank.git
$ pip install -r requirements.txt
# For all of my urls within ./backlog/
$ python3 app.py -h
# For all urls you want to feed it
$ python3 app.py -i yoururls.txt -o path/to/download/to
```
